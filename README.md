# PCS

Welcome to the official website repository for PCS. This website showcases our range of high-quality property maintenance services with a focus on "No Nonsense" user experience.

## Services Included

*   **Lawn Surgeon:** Premium lawn maintenance and restoration.
*   **Omega Window Cleaning:** Residential window cleaning (Exterior, Interior, and Signature Service).

## Technologies Used

*   **HTML5**
*   **Tailwind CSS** (via CDN)
*   **JavaScript** (Vanilla)
*   **Google Fonts** (Cormorant Garamond)
*   **Image Comparison Slider** (for galleries)
*   **JSON-LD Schema** (for Local Business SEO)

## Setup

This is a static site. To view it, simply open `index.html` in your web browser.
